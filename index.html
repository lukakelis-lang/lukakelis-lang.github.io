📄 LV15 — Povezivanje HTML-a i CSS-a, vrste i sintaksa CSS-a

Luka Keliš i Karlo Kasumović · 3.B · 14.1.2026.

Priprema za vježbu:

  • HTML opisuje strukturu i sadržaj web stranice, a CSS njezin izgled i oblikovanje.
  • CSS je kratica za Cascading Style Sheets (kaskadne stilske tablice).
  • CSS je stvorila organizacija W3C radi odvajanja sadržaja od izgleda.
  • Preporučeni način je vanjska CSS datoteka (.css) jer omogućuje oblikovanje više HTML stranica i lakše održavanje.

Tri načina ubacivanja CSS-a:

  • Eksterni (vanjski) CSS — preporučeno! Jedna .css datoteka za cijelo sjedište. Uključuje se <link> tagom u <head>.
  • Interni CSS — definira se unutar <style> taga u <head> sekciji dokumenta.
  • Linijski (inline) CSS — dodaje se direktno na element pomoću atributa style. Preporučuje se samo iznimno.

Primjer CSS pravila (sintaksa):

h1 {
  color: red;
  text-align: center;
}
/* Selektor: h1 | Svojstvo: color, text-align | Vrijednost: red, center */

📄 LV16 — Vrste selektora i njihova primjena

Luka Keliš i Karlo Kasumović · 3.B · 19.1.2026.

Pet kategorija CSS selektora:

  • Jednostavni selektori — odabiru elemente prema imenu, id-u ili klasi
  • Složeni selektori — odabiru elemente prema specifičnim odnosima
  • Selektori pseudo-klasa — odabiru elemente prema stanju (npr. :hover)
  • Selektori pseudo-elemenata — stiliziraju dio elementa (npr. ::before)
  • Selektori atributa — odabiru prema atributu ili vrijednosti atributa
/* Selektor elementa */
p { text-align: center; color: red; }

/* ID selektor */
#odlomak1 { text-align: left; color: blue; }

/* Selektor klase */
.centrirano { text-align: center; color: green; }

/* Samo p elementi s klasom centrirano */
p.centrirano { text-align: center; color: green; }

📄 LV17 — CSS pozadine i uređivanje

Luka Keliš i Karlo Kasumović · 3.B · 28.1.2026.

Načini zadavanja boja u CSS-u:

  • Naziv boje → red, blue, green
  • HEX → #ff0000
  • RGB → rgb(255, 0, 0)
  • RGBA → rgba(255, 0, 0, 0.5) — s prozirnošću (0=prozirno, 1=neprozirno)
  • HSL / HSLA → hsl(0, 100%, 50%)

Vježba — stranica s citatima (HTML + CSS):

<!DOCTYPE html>
<html lang="hr">
<head>
  <meta charset="UTF-8">
  <meta name="author" content="Luka Keliš i Karlo Kasumović">
  <title>Citati</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <section class="quotes">
    <blockquote class="quote quote-1">
      <p>"Mudrost je znanje koje dolazi s iskustvom."</p>
      <footer>- Nepoznat autor</footer>
    </blockquote>
    <blockquote class="quote quote-2">
      <p>"Ljubav je najvažnija stvar na svijetu."</p>
      <footer>- Nepoznat autor</footer>
    </blockquote>
  </section>
</body>
</html>

CSS svojstva pozadine: background-color, background-image, background-repeat, background-position, background-size.

📄 Esej: Važnost sporta u životu mladih

Predmet: Hrvatski jezik · Datum: rujan 2025.

Sport ima iznimno važnu ulogu u životu mladih ljudi. Kroz redovitu tjelesnu aktivnost ne razvijamo samo fizičke sposobnosti, već i karakter, disciplinu i sposobnost rada u timu.


Osobno se bavim street workoutom koji zahtijeva veliku dozu samodiscipline i strpljenja. Nema trošite ni opreme ni novca — samo vlastita snaga i volja. Upravo ta jednostavnost čini ga dostupnim svima.


Studije pokazuju da mladi koji se redovito bave sportom postižu bolje rezultate u školi, imaju manje problema s mentalnim zdravljem i razvijaju bolje socijalne vještine. Sport uči kako prihvatiti poraz i ustati nakon njega.


Zaključno, sport nije samo zabava — to je investicija u budućnost. Svaki mladi čovjek trebao bi pronaći sportsku aktivnost koja mu odgovara i baviti se njome redovito.

LV12 — Istraživanje

Tema istraživanja: Utjecaj društvenih mreža na mlade

Predmet: SJWP · Školska godina 2024./2025.

Uvod
Društvene mreže postale su nezaobilazan dio svakodnevnog života mladih. Instagram, TikTok, YouTube i Snapchat koriste mladi od 10. do 20. godine života svakodnevno, prosječno 3–6 sati dnevno.
Metodologija

Istraživanje je provedeno metodom online ankete. Sudjelovalo je 30 učenika Tehničke škole Ruđera Boškovića, razredi 2. i 3. Anketa se sastojala od 10 pitanja o vremenu korištenja, omiljenim platformama i utjecaju na produktivnost.

Rezultati
Platforma Koriste svaki dan Prosječno vrijeme Utjecaj na učenje
YouTube87%2.5 h/danUglavnom neutralno
Instagram73%1.5 h/danMalo negativno

Zaključak
Istraživanje je pokazalo da mladi provode značajnu količinu vremena na društvenim mrežama. Dok YouTube ima uglavnom neutralan ili pozitivan utjecaj (učenje kroz videe), TikTok i Instagram negativno utječu na koncentraciju i produktivnost. Preporuka je ograničiti korištenje na 1-2 sata dnevno i koristiti platforme svjesno.
Moji zaključci
  • Mladi nisu svjesni koliko vremena provedu na mrežama
  • Kratki video formati (Reels, TikTok) smanjuju pažnju
  • YouTube može biti i edukativna platforma ako se koristi ciljano
  • Potrebna je medijska pismenost u školama
LV13a — Zapisnik radionice

Radionica: Izrada osobnog mrežnog sjedišta

Datum: 2025. · Predmet: SJWP

Tijek radionice

1. dio — Planiranje strukture

Na početku radionice definirali smo strukturu stranice — koje kategorije će imati, koliko podstranica i kakav dizajn koristiti. Dogovorena je navigacijska shema s glavnim stranicama.

2. dio — Postavljanje HTML temelja

Kreiran je osnovni HTML dokument s ispravnom DOCTYPE deklaracijom, meta tagovima i navigacijom. Koristili smo semantičke HTML5 elemente: <nav>, <main>, <section>, <footer>.

<!DOCTYPE html>
<html lang="hr">
<head>
  <meta charset="UTF-8">
  <title>Osobna stranica</title>
</head>
<body>
  <nav>...</nav>
  <main>...</main>
</body>
</html>

3. dio — CSS stilizacija

Dodani su CSS stilovi za boje, fontove, raspored i responzivnost. Korišten je CSS Flexbox za raspored elemenata u navigaciji i karticama.

4. dio — Sadržaj stranica

Svaki učenik je dodao vlastiti sadržaj — tekst, tablice, forme i liste. Sav sadržaj je organiziran po kategorijama s jasnom navigacijom.

Naučeno na radionici
  • Struktura web sjedišta — planiranje prije kodiranja
  • Semantički HTML — <nav>, <section>, <article>, <footer>
  • CSS Flexbox — postavljanje elemenata u red
  • Responzivnost — prilagodba za mobilne uređaje
  • Objava na internetu — hosting i FTP upload
Zaključak
Radionica LV13a bila je koristan praktičan sat na kojemu smo naučili kako se planira i izrađuje osobno web sjedište. Stekli smo iskustvo s HTML-om i CSS-om kroz pravu primjenu, a ne samo teoriju.
